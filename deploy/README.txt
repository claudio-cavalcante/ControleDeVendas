# GDV - Gestão de Vendas

Controle de estoque de Supermercado. 

## Introdução

Programa de Controle de Estoque de produtos de supermercado. 

Com este programa de controle de estoque você poderá controlar todas as entradas e saídas de produtos na sua loja. Manter o seu estoque em dia, saiba quais produtos você deve comprar, quais você cadastrou no sistema etc. Controle de estoque eficiente multiplatatorma.

O programa tem as seguites funcionalidades: 

	*    Controle de Visibilidade, somente gerente emite relatórios.	
	*    Emissão de relatório de Produtos e Estoque.
	*    Entradas/Compras
	*    Saídas/Vendas
	*    Estoque Atual e Cadastro de Produtos


### Pré-requisitos

É necessário ter instalado o JRE (Java RUNTIME Environment): 

Download: http://www.oracle.com/technetwork/java/javase/downloads/index.html


### Instalando

Windows: 

Clique duplo do mouse no arquivo GDV.bat , ou abra o prompt ou shell e digite:

java -jar GDV.jar


Linux: 

Abra o shell e digite:

java -jar GDV.jar
